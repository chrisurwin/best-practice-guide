## Standalone
---
### Characteristics

 * Simple, low-cost environment for development and training 
 * No resiliency to host failure 
 * Low performance 

### Instructions

1. Create a Kubernetes environment 
2. Add 1 host with >=1 CPU and >=4GB RAM
